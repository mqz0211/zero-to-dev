# YouTube Channels

Curated, not exhaustive — each entry earns its place by teaching
something the others don't.

## General programming & CS fundamentals
- **freeCodeCamp.org** — full-length, free courses on nearly every
  topic in this repo, from "Python for beginners" to "Linux for beginners"
- **CS50 (Harvard)** — the lecture recordings for Harvard's famous intro
  CS course, free on YouTube
- **NetworkChuck** — networking and Linux, energetic and beginner-friendly

## Math & AI/ML intuition
- **3Blue1Brown** — the best visual explanations of linear algebra,
  calculus, and neural networks that exist, free
- **StatQuest with Josh Starmer** — statistics and machine learning
  concepts explained clearly, one idea at a time

## Web development
- **Fireship** — extremely dense, fast-paced overviews; great for
  "what is this technology and why would I use it" before going deep
- **Web Dev Simplified** — clear, focused tutorials on JavaScript and React

## Cybersecurity
- **John Hammond** — CTF walkthroughs and security concepts explained
  as he solves them
- **The Cyber Mentor** — ethical hacking and pentesting fundamentals

## Game development
- **Brackeys** (archived, but still excellent) and **Sebastian Lague**
  — Unity fundamentals and the math/logic behind game systems

## Cloud & DevOps
- **TechWorld with Nana** — Docker, Kubernetes, and cloud concepts
  explained clearly for beginners
