# Websites & Tools

## Practice & learning platforms
| Site | Best for | Cost |
|---|---|---|
| [freeCodeCamp.org](https://www.freecodecamp.org) | Full curricula: web dev, Python, data science | Free |
| [CS50.harvard.edu](https://cs50.harvard.edu) | Rigorous intro CS fundamentals | Free (audit) |
| [The Odin Project](https://www.theodinproject.com) | Full-stack web development, project-based | Free |
| [NeetCode](https://neetcode.io) | Data structures & algorithms interview prep | Free tier |
| [Exercism](https://exercism.org) | Language-specific coding practice with mentor feedback | Free |
| [Kaggle Learn](https://www.kaggle.com/learn) | Short, hands-on data science/ML courses | Free |
| [OverTheWire](https://overthewire.org/wargames/) | Linux & security "wargames" | Free |
| [PortSwigger Web Security Academy](https://portswigger.net/web-security) | Web security, hands-on labs | Free |
| [TryHackMe](https://tryhackme.com) | Guided cybersecurity learning paths | Free tier |
| [roadmap.sh](https://roadmap.sh) | Visual, interactive career roadmaps (great companion to this repo) | Free |

## Reference documentation (bookmark these)
- [MDN Web Docs](https://developer.mozilla.org) — the definitive HTML/CSS/JS reference
- [DevDocs](https://devdocs.io) — searchable docs for dozens of languages/frameworks in one place
- [Regex101](https://regex101.com) — build and test regular expressions with live explanations

## Tools you'll actually use
- **[VS Code](https://code.visualstudio.com)** — the default code editor;
  free, extensible, works for every language in this repo
- **[GitHub](https://github.com)** — where your code lives and your
  portfolio is built (see [`CAREERS/github-guide.md`](../CAREERS/github-guide.md))
- **[Postman](https://www.postman.com)** — testing APIs without writing a client
- **[Figma](https://www.figma.com)** — free tier, useful if you ever
  design a UI before building it

## Communities (ask questions here)
- **r/learnprogramming**, **r/cscareerquestions** — Reddit
- Language/framework-specific Discord servers (most major frameworks
  have an official one — check their docs site)
- Your local university/school's coding club, if one exists — in-person
  accountability is underrated
