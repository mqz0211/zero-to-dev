# Free Courses

> Curated, not exhaustive — every course listed here is free, still
> actively maintained, and genuinely well-regarded, not just SEO'd.
> Each entry includes what track it fits and roughly how deep it goes.

## Foundations / CS fundamentals

| Course | Provider | Best for |
|---|---|---|
| CS50: Introduction to Computer Science | Harvard (edX/YouTube) | The single best general starting point for anyone new to CS |
| The Odin Project | Community | Full-stack web development, project-heavy |
| freeCodeCamp curriculum | freeCodeCamp | Broad, structured, certificate on completion |
| Teach Yourself CS | teachyourselfcs.com | Self-directed CS fundamentals reading list |

## Programming languages

| Course | Provider | Best for |
|---|---|---|
| Python for Everybody | University of Michigan (Coursera, audit free) | Python from zero |
| CS50's Introduction to Programming with Python | Harvard | Python with a CS lens |
| JavaScript.info | Community | The most thorough free JS reference/course |

## Web Development

| Course | Provider | Best for |

## Cybersecurity

| Course | Provider | Best for |

## AI / Machine Learning

| Course | Provider | Best for |
|---|---|---|

| Practical Deep Learning for Coders | fast.ai | Hands-on, code-first deep learning |
| CS231n / CS224n lecture notes | Stanford (public) | Deeper dives into vision/NLP once you have basics |

## A note on freshness

Free course platforms change frequently — a link that worked last year
may now be paywalled or discontinued. If you find a dead link, please
open an issue or PR — see [`CONTRIBUTING.md`](../CONTRIBUTING.md).
